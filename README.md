# 📦 Inventario Para Tiendas Skillnest 

## Descripción

Proyecto desarrollado en Java para gestionar el inventario de una tienda de productos electrónicos, aplicando conceptos fundamentales de Programación Orientada a Objetos (POO)
como herencia, encapsulamiento, polimorfismo y reutilización de código.

---

## 🎯 Objetivos

- Comprender y aplicar el concepto de herencia en la programación orientada a objetos.
- Practicar la creación de una clase base y clases derivadas utilizando herencia.
- Implementar constructores y métodos en las clases derivadas para extender o sobrescribir el comportamiento de la clase base.
- Realizar pruebas para verificar el correcto funcionamiento de las clases y sus métodos.
- Reforzar la comprensión de los principios de encapsulamiento y reutilización del código mediante el uso de herencia.

---

## 🛠️ Tecnologías Utilizadas
- Java
- Visual Studio Code (VSC)
- Programación Orientada a Objetos (POO)

---

## 🧠 Conceptos Aplicados
- Herencia
- Encapsulamiento
- Polimorfismo
- Sobrescritura de métodos
- Sobrecarga de constructores
- Colecciones (ArrayList)
- Gestión de inventario
- Programación Orientada a Objetos (POO)
